# 🎯 Jogo do Número Secreto

Projeto desenvolvido durante os cursos da Alura com o objetivo de praticar lógica de programação e JavaScript.

## 📋 Descrição

O jogo consiste em adivinhar um número secreto gerado aleatoriamente pelo sistema. A cada tentativa, o jogador recebe dicas informando se o número secreto é maior ou menor que o valor informado.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 🎮 Como jogar

1. Abra o arquivo `index.html` no navegador.
2. Digite um número no campo indicado.
3. Clique no botão **Descobrir** (ou **Adivinhar**, dependendo da versão).
4. Continue tentando até descobrir o número secreto.

## 📚 Objetivos do projeto

- Praticar lógica de programação.
- Utilizar estruturas condicionais e de repetição.
- Trabalhar com funções em JavaScript.
- Manipular elementos da página utilizando o DOM.
